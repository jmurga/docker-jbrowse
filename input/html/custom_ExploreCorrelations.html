This section contains a tool that allows generating custom interactive plots to explore correlations among different genomic and geographical features estimated for each population.

When the user clicks on a specific population, its value is removed from the graph and the correlation statistics are re-calculated. Multiple points can be removed at once by selecting the corresponding area in the graph and clicking the “Toggle points” button. To reset all points, click the “Reset points” button.

This application has been developed using the Shinyapps framework for R.
